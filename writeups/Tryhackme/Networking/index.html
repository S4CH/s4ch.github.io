Kinda like a street address, just cooler.

In a manner similar to streets and homes, computers and their respective communication networks must have a way to address their 'mail'. In the following set of questions, we'll investigate the various types of IPv4 addresses. Here's a reference guide for that various types of IP addresses:

IP Address Classes

Private Address Space

5

  1. Which type is for research? *Looking for a letter rather than a number here

E

  1. How many private address ranges are there?

3

  1. Which private range is typically used by businesses?

A

  1. There are two common default private ranges for home routers, what is the first one?

192.168.0.0

  1. How about the second common private home range?

192.168.1.0

  1. How many addresses make up a typical class C range? Specifically a /24

256

  1. Of these addresses two are reserved, what is the first addresses typically reserved as?

Network

  1. The very last address in a range is typically reserved as what address type?

Broadcast

  1. A third predominant address type is typically reserved for the router, what is the name of this address type?

Gateway

  1. Which address is reserved for testing on individual computers?

127.0.0.1

  1. A particularly unique address is reserved for unroutable packets, what is that address? This can also refer to all IPv4 addresses on the local machine.

0.0.0.0

Binary to Decimal

Binary conversion is essential to understand in order to properly manage computer networks. An IPv4 address consists of 32 bits split up into four sections of eight bits. For example, the address 192.168.1.12 translates to this:

11000000 10101000 00000001 00001100
